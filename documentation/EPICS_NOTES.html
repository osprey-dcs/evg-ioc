Introduction

This IOC application uses a LEEP driver to communicate with Osprey Event Generator and Event Receiver node.  The records mentioned in the following sections have names beginning with a $(P) macro.

Build Information

The time at which the firmware and software were built is contained in the per-node stringin records Date:FW-I and Date:SW-I, respectively.

Event Generator and Event Receiver Records

Event generator and event receiver records are described in the documentation for their respective support modules.

Internal Records

Records with names ending in an underscore are intended for internal use by the IOC and should not be written by any other clients.
